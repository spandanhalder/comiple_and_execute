# Python Automated Script For Compile and Execute For Text Editors.
![Preview](SCREENSHOTS/preview.gif)

## Features
-	This is Python automated script which helps to compile and execute the source code by a single keypress.
-	First it identifies the extension and then runs the predefined commands to compile and execute.
-	On successful compilation shows the compilation time taken else an ASCII Error Art is produced.
-	Always opens up new terminal execution window. 


## Instructions:
Run start.bat with full-path including filename.
```
C:\Users\Username>start.bat path\with\filename.extension
```
